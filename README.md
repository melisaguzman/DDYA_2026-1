# DDYA_2026-1
En el contenido se encuentran los trabajos semana por semana de la asignatura Diseño de Datos y Algoritmos 


# Semana 1
En la semana 1 se puede evidenciar las pruebas técnicas para el diagnóstico en las cuales se encuentran dos códigos para la resolución de dos problemas, junto a sus respectivos diseños.

# Semana 2
En la semana dos introdujimos el tema de diseño, en el cual se puede evidenciar la evolución del diseño en el algoritmo planteado, así mismo, encontrará ejercicios del análisis de tiempo de ejecucón de un código.
